Apa itu CSS?

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan dan tata letak elemen-elemen HTML. CSS memungkinkan kita untuk mengubah warna, ukuran, jarak, dan banyak aspek visual lainnya dari elemen di halaman web.

Penggunaan CSS untuk Mengatur Tampilan Elemen HTML

CSS dapat diterapkan ke elemen HTML untuk mengubah tampilannya. Berikut contoh sederhana pengaturan warna latar belakang dan ukuran font menggunakan CSS:

body {
  background-color: #f0f0f0;
  font-size: 16px;
}

Class dalam CSS untuk Mengelompokkan Gaya

CSS menggunakan class untuk mengelompokkan elemen-elemen dengan gaya yang sama. Contoh penggunaan class:

.highlight {
  color: white;
  background-color: #3498db;
  padding: 10px;
  border-radius: 5px;
}

Ini adalah contoh penggunaan class CSS.

Division <div> untuk Pengelompokan Konten

Elemen <div> digunakan untuk mengelompokkan elemen-elemen HTML lain menjadi satu blok. Ini sangat berguna untuk menyusun konten dalam halaman web.

<div class="container">
  <h2>Judul Section</h2>
  <p>Ini adalah isi dari section.</p>
</div>
