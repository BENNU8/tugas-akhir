Pengenalan HTML

Apa itu HTML?

HTML (HyperText Markup Language) adalah bahasa standar yang digunakan untuk membuat dan merancang halaman web. HTML memungkinkan pengembang untuk mendefinisikan struktur konten menggunakan elemen-elemen seperti heading, paragraf, tautan, gambar, dan banyak lagi.

Struktur Dasar HTML

Dokumen HTML terdiri dari elemen-elemen yang dibuka dengan tag pembuka dan ditutup dengan tag penutup. Contoh struktur dasar HTML:

<!DOCTYPE html>
<html>
    <head>
        <title>Judul Halaman</title>
    </head>
    <body>
        <h1>Halo Dunia!</h1>
    </body>
</html>

Basic HTML Tags

  • <html>: Menandakan awal dan akhir dokumen HTML.
  • <head>: Bagian untuk metadata seperti judul halaman dan referensi ke file eksternal.
  • <body>: Menampung semua konten halaman yang terlihat oleh pengguna.
  • <h1> hingga <h6>: Heading atau judul dengan berbagai ukuran.

Format Teks dan Karakter Khusus

HTML menyediakan elemen untuk memformat teks, seperti <b> untuk teks tebal, <i> untuk teks miring, dan <u> untuk teks bergaris bawah.

Untuk karakter khusus, seperti tanda copyright atau simbol lainnya, gunakan entitas HTML seperti &copy; untuk Â©.

Menampilkan Gambar, Suara, dan Video

HTML memungkinkan untuk menampilkan media menggunakan elemen berikut:

  • <img>: Menampilkan gambar.
  • <audio>: Menyisipkan file audio.
  • <video>: Menyisipkan file video.

Contoh penggunaan:

<img src="image.jpg" alt="Deskripsi gambar">
<audio controls><source src="audio.mp3" type="audio/mp3"></audio>
<video controls><source src="video.mp4" type="video/mp4"></video>

Hyperlink dengan <a>

HTML memungkinkan untuk membuat hyperlink menggunakan elemen <a>. Berikut contoh penggunaannya:

<a href="https://www.example.com">Kunjungi Website</a>

Daftar, Penomoran, dan List

HTML menyediakan elemen untuk membuat daftar:

  • <ul> dan <li>: Daftar tak berurut (bullet).
  • <ol> dan <li>: Daftar berurut (nomor).

Pembuatan Tabel

HTML memungkinkan pembuatan tabel menggunakan elemen-elemen seperti <table>, <tr>, dan <td>. Berikut contoh tabel:

<table>
  <tr>
    <th>Nama</th>
    <th>Usia</th>
  </tr>
  <tr>
    <td>John</td>
    <td>25</td>
  </tr>
</table>

Formulir dan Input

HTML memungkinkan pembuatan formulir dengan elemen-elemen seperti <form>, <input>, dan <textarea>:

<form action="submit.php">
  <label>Nama:</label>
  <input type="text" name="name">
  <textarea name="message"></textarea>
  <input type="submit" value="Kirim">
</form>

Frame dan Iframe

Anda dapat menyisipkan halaman lain dalam halaman HTML menggunakan <iframe>:

<iframe src="https://www.example.com" width="600" height="400"></iframe>
