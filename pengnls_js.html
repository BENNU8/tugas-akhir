Pengenalan JavaScript

Dialog Box: alert(), prompt(), confirm()

Dialog box digunakan untuk berinteraksi dengan pengguna. Berikut adalah contoh cara kerjanya:

alert('Halo, Selamat datang di JavaScript!');
const nama = prompt('Siapa nama Anda?');
const apakahSetuju = confirm('Apakah Anda suka JavaScript?');

Tipe Data dan Operator

JavaScript mendukung berbagai tipe data seperti string, angka, dan boolean, serta operator untuk melakukan operasi pada data:

let angka = 5;
let teks = 'Halo';
let aktif = true;
let jumlah = angka + 10; // Operator Aritmatika

Struktur Kendali: if, else, switch

Pernyataan kondisi seperti if, else, dan switch digunakan untuk mengontrol alur program:

if (angka > 5) {
  console.log('Angka lebih besar dari 5');
} else {
  console.log('Angka kurang dari atau sama dengan 5');
}

switch (teks) {
  case 'Halo':
    console.log('Salam');
    break;
  default:
    console.log('Tidak ada kecocokan');
}

Pengulangan: for, while

Loop seperti for dan while digunakan untuk mengeksekusi kode berulang kali:

for (let i = 0; i < 5; i++) {
  console.log(i);
}

let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}

Fungsi dan Array

Fungsi dan array adalah bagian penting dalam JavaScript. Fungsi memungkinkan kita untuk menggunakan kode yang sama berkali-kali, dan array digunakan untuk menyimpan beberapa nilai:

function sapa(nama) {
  console.log('Halo ' + nama);
}

const buah = ['Apel', 'Pisang', 'Ceri'];
buah.forEach(buah => console.log(buah));
